# 🛍️ Customer Shopping Behaviour Analysis

An end-to-end data analysis project that explores customer shopping patterns using **Python**, **SQL Server**, and **Power BI**. The project covers data ingestion from CSV into a SQL Server database, SQL-based analysis across 10 business questions, and an interactive Power BI dashboard for visual exploration.

## 📊 Dataset

The dataset (`shopping_behaviour`) contains retail transaction records with the following key fields:

| Field | Description |
|---|---|
| `customer_id` | Unique customer identifier |
| `gender` | Male / Female |
| `age_group` | Age bracket of the customer |
| `category` | Product category |
| `item_purchased` | Specific product bought |
| `purchase_amount_(usd)` | Transaction value in USD |
| `review_rating` | Customer rating (1–5) |
| `subscription_status` | Subscribed / Not Subscribed |
| `shipping_type` | Standard / Express / etc. |
| `discount_applied` | Yes / No |
| `previous_purchases` | Number of past purchases |

## 🚀 Getting Started

### Prerequisites

- Python 3.8+
- Microsoft SQL Server (local instance)
- ODBC Driver 17 for SQL Server
- Power BI Desktop (to view `.pbix` file)
- Jupyter Notebook

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Priyanshu140/Customer-Shopping-Behaviour-Analysis.git
   cd Customer-Shopping-Behaviour-Analysis
   ```

2. **Install Python dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Load data into SQL Server**

   Run the Jupyter notebook (`analysis.ipynb`). It reads the CSV and loads it into SQL Server using Windows Authentication — no password needed:
   ```python
   engine = create_engine(
       "mssql+pyodbc://localhost/customer_shopping_behaviour"
       "?driver=ODBC+Driver+17+for+SQL+Server&trusted_connection=yes"
   )
   df.to_sql("shopping_behaviour", engine, if_exists="replace", index=False)
   ```

4. **Run SQL Analysis**

   Open `Customer_Query1.sql` in SQL Server Management Studio (SSMS) and execute the queries against the `customer_shopping_behaviour` database.

5. **View the Dashboard**

   Open `customer_behavior_dashboard.pbix` in Power BI Desktop.

## 🔍 SQL Analysis — Business Questions

The SQL file answers 10 key business questions:

| # | Question |
|---|---|
| Q1 | What is the total revenue generated by male vs female customers? |
| Q2 | Which customers used a discount but still spent more than the average purchase amount? |
| Q3 | What are the top 5 products with the highest average review rating? |
| Q4 | How does average purchase amount compare between Standard and Express shipping? |
| Q5 | Do subscribed customers spend more? (Average spend & total revenue comparison) |
| Q6 | Which 5 products have the highest percentage of discount-applied purchases? |
| Q7 | Segment customers into **New**, **Returning**, and **Loyal** based on previous purchases |
| Q8 | What are the top 3 most purchased products within each category? |
| Q9 | Are repeat buyers (5+ previous purchases) more likely to subscribe? |
| Q10 | What is the revenue contribution of each age group? |

### Customer Segmentation Logic (Q7)
```sql
CASE
    WHEN previous_purchases = 1              THEN 'New'
    WHEN previous_purchases BETWEEN 1 AND 10 THEN 'Returning'
    ELSE                                          'Loyal'
END
```
